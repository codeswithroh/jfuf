# JFUF - Just F*cking Use Flow

A modern landing page for Flow blockchain - the only L1 built for consumer apps.

## About

JFUF showcases the power of Flow blockchain with:
- Gasless UX for seamless user experience
- Resource-oriented security model
- 1-second finality for instant transactions
- Built specifically for consumer applications

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Deployment**: Optimized for Vercel

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Build & Deploy

Build for production:

```bash
npm run build
```

Deploy to Vercel:

```bash
npm run deploy
```

## Features

- **Responsive Design**: Optimized for all screen sizes
- **Modern UI**: Clean, bold design with smooth animations
- **Fast Loading**: Optimized build with static generation
- **SEO Ready**: Proper meta tags and structured data
